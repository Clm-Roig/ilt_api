# Get started

## Dependencies

- Python 3.7+
- Django
- Django Rest Framework
- PostGreSQL

## Installation

```sh
pip install -r requirements.txt
```

```sh
cd ilt_api
./manage.py runserver
```

## Tests

```sh
./manage.py test
```
